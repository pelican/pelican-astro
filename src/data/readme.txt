data
====
This folder contains definitions of the data classes used throughout PELICAN.

List of data blobs used in the PELICAN pipeline:
------------------------------------------------
- Antenna position (x,y,z) data.
- Visibility position (u,v,w) data.
- Visibility data.
- Flag table data (mirrors visibility data structure).
- Antenna complex gain data (mirrors visibility data structure).
- Image data.
- Frequency list data.
